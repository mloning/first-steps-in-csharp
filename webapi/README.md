# Web API

Run the API:

```bash
dotnet run --urls https://localhost:3000
```

Open the docs:

```bash
open https://localhost:3000/swagger
```

Test the API:

```bash
curl -X 'GET' \
  'https://localhost:3000/WeatherForecast' \
  -H 'accept: text/plain'
```
